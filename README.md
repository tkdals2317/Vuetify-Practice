
## vue-study

## Motivation 

프론트엔트 공부를 시작해봅시다!   

최근 급 부상하고 있는 Vue의 Vuetify는 Vue 생태계에서 가장 활동적이며 다양한, 완벽한 기능을 갖춘 개발 프레임 워크입니다. 

쉽게 말해 vue에서 사용할 수 있는 용이한 UI toolkit이라고 할 수 있습니다.

그래서 Vuetify를 공부해야할 필요성을 느꼈고 공부하면서 정리한 자료를 저장하기 위한 레포지토리 입니다.
